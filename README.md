# TypeSpeed

A minimal typing speed test web app inspired by [keybr.com](https://www.keybr.com/). Built with vanilla HTML, CSS, and JavaScript — no dependencies.

## Features

- Real-time WPM and accuracy tracking
- Four time modes: 15s, 30s, 60s, 120s
- Paragraph-style text display with smooth line-by-line scrolling
- Character-level feedback — correct, incorrect, and cursor states
- Animated countdown ring that turns red in the final 5 seconds
- Progress bar tracking position through the text
- Results summary with WPM, accuracy, correct characters, and errors
- Auto-extends word pool so you never run out of text
- Claude color scheme with iOS-style UI

## Structure

```
├── index.html   — markup
├── style.css    — styles and CSS variables
└── app.js       — all logic
```

## Usage

Open `index.html` directly in a browser. No build step or server required.

Click anywhere on the text card or press any key to begin. The timer starts on your first keystroke.
